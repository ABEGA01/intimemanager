GUIDE DU DÉVELOPPEUR - INTIMEMANAGER
===================================

1. STRUCTURE DU PROJET
---------------------
/
├── docs/               # Documentation
├── images/            # Images et logos
├── tcpdf/             # Bibliothèque TCPDF
├── Connexion.php      # Configuration de la base de données
├── index.php          # Page d'accueil
├── login.php          # Page de connexion
├── ListeArticles.php  # Gestion des articles
├── ListeVentes.php    # Gestion des ventes
├── ListeFournisseurs.php # Gestion des fournisseurs
├── Rapports.php       # Rapports et statistiques
└── Imprimer*.php      # Fichiers d'impression

2. CONFIGURATION DE L'ENVIRONNEMENT
----------------------------------
2.1 Prérequis
- PHP 7.4 ou supérieur
- MySQL 5.7 ou supérieur
- Serveur web (Apache/Nginx)
- Extensions PHP requises :
  * PDO
  * PDO_MySQL
  * GD
  * mbstring

2.2 Installation
1. Cloner le dépôt
2. Configurer la base de données dans Connexion.php
3. Créer la base de données et importer le schéma
4. Configurer les permissions des dossiers

3. BASE DE DONNÉES
------------------
3.1 Structure
- Tables principales :
  * article
  * vente
  * detail_vente
  * utilisateur
  * fournisseur
  * categorie

3.2 Relations
- vente -> utilisateur (IDUTILISATEUR)
- detail_vente -> vente (IDVENTE)
- detail_vente -> article (IDARTICLE)
- article -> categorie (IDCATEGORIE)
- article -> fournisseur (IDFOURNISSEUR)
